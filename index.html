Custom Linux GRR Scheduler

A full sched_class implementation wired into linux 6.14 — a Guaranteed Round Robin policy built to deliver predictable execution windows for latency-sensitive tasks. I extended the core scheduling structures, owned the per-CPU runqueue, and made every migration decision both group-aware and rate-limited.

  • Full RR pipelineenqueue · pick_next · task_tick
  • balance_grrpush/pull, 500ms throttle
  • grr_performance_maskDEFAULT vs PERFORMANCE groups
  • double_lock_balancesafe cross-CPU migration
static void enqueue_task_grr(struct rq *rq,
                            struct task_struct *p,
                            int flags) {
    struct grr_rq *grr_rq = &rq->grr;
    if (!p->grr.time_slice)
        p->grr.time_slice = GRR_TIMESLICE;
    list_add_tail(&p->grr.run_list, &grr_rq->queue);
    grr_rq->nr_running++;
}

MiniJava Static Checker

The semantic-analysis phase of a complete MiniJava compiler. JavaCC produces the parse tree, JTB generates the visitor scaffolding, and a hand-written two-pass visitor enforces scoping, type safety and inheritance validity before code generation. It catches the subtle errors — override signature mismatches, non-assignable arguments, broken vtable layouts — that a single-pass checker misses.

  • Symboltlb (Pass 1)class table + offsets
  • TypeCheckVisitor (Pass 2)subtyping · assignability
  • vtable buildoverride slot reuse
  • offset layoutint=4 · bool=1 · ref=8
// inheritance + method resolution
Methodinfo resolveMethod(String recv,
                            String name,
                            List<Types> args) {
    for (Classinfo c = classTable.get(recv);
         c != null; c = classTable.get(c.parent)) {
        for (Methodinfo m : c.methods)
            if (m.name.equals(name)
                && m.params.equals(args)
                && assignableAll(args, m.params))
                return m;
    }
    reportError("method not found");
    return null;
}

jms_coord — Multi-Threaded Job Server

A high-concurrency Job Management System coordinator written in pure C. A pool of worker threads waits on a condition variable, handlers accept jobs over a text-based protocol, and children are launched with fork + execvp with their output redirected to per-job directories — the textbook monitor pattern, made deadlock-free.

  • Monitor patternmutex + cond_non_empty
  • fork / execvpper-job output dir
  • read_line / write_allno partial TCP reads
  • graceful shutdowncond_broadcast + join

Tedy — Event Platform w/ BMF Recommendations

A scalable event-discovery platform built to handle thousands of concurrent registrations. Angular on the front, NestJS + Prisma + PostgreSQL underneath, and a recommendation engine I trained from scratch: a Biased Matrix Factorization model with SGD, blended across bookings, views and negative signals. Booking itself is concurrency-safe via Serializable isolation and a SELECT … FOR UPDATE row lock — no oversold tickets.

WikiArt Image Classification (FNN & CNN)

A university deep-learning study on classifying paintings into four art-movement classes (Baroque, Impressionism, Cubism, Abstract Expressionism) from the WikiArt dataset at 32×32 resolution. I trained a flat-pixel FNN baseline, then designed a 4-layer CNN and ablated padding + pooling + ReLU one piece at a time — the convolutional structure turned out to matter far more than parameter count.

  • FNN baseline3 layers · F1 0.4943
  • CNN §2.31.06M params · F1 0.5539
  • Optimizer sweepSGD · Adam · AdamW · RMSprop
  • Reproducibleseed=42 · Kaggle T4×2
# 4-layer CNN (best configuration)
class CNN(nn.Module):
    def __init__(self):
        self.conv1 = Conv2d(3,  16, k=5, padding=2)
        self.conv2 = Conv2d(16, 32, k=5, padding=2)
        self.conv3 = Conv2d(32, 64, k=5, padding=2)
        self.conv4 = Conv2d(64,128, k=5, padding=2)
        self.pool  = MaxPool2d(2)
    def forward(self, x):
        x = relu(self.pool(self.conv1(x)))
        x = relu(self.pool(self.conv2(x)))
        x = relu(self.pool(self.conv3(x)))
        x = relu(self.pool(self.conv4(x)))
        return self.fc(x.flatten(1))
